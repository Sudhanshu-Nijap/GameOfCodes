# StegoReveal — LSB Steganography Suite

StegoReveal is an API and UI suite that hides and reveals secret messages inside PNG and JPG images using Least Significant Bit (LSB) steganography.

## Project Structure

```
Game_Of_Codes/
├── api.py                  (FastAPI backend Python server)
├── Steganography/          (Core logic and Desktop GUI script)
└── WebStego/               (Web frontend HTML, CSS, JS)
```

## Core Logic and Features

StegoReveal modifies the least significant bit of image pixels to embed hidden data seamlessly without altering visual fidelity.

The API exposes an encode endpoint (`POST /encode`) that receives an image and a message, appends a strict "###END###" delimiter to the message, and sequentially interleaves the binary text into the RGBA color channels of the image. The resulting image is streamed back as a downloadable file.

The API also exposes a highly resilient decode endpoint (`POST /decode`) designed to extract text even if the method of encoding is unknown. It simultaneously scrapes pixel data across six different strategies, including interleaved RGBA, interleaved RGB, and isolated Red, Green, Blue, or Alpha individual channels. 

During extraction, the server first attempts to find the strict "###END###" boundary. If this delimiter is located in any extracted stream, the backend immediately halts processing and serves the guaranteed match to the frontend.

If no strict boundary is detected, the API assumes the image was encoded using third-party tools or raw noisy data. It applies aggressive string cleaning algorithms to strip repeating garbage characters generated by unused pixel noise. By leveraging regex filtering, the decoder then dynamically isolates standalone URLs and sequence numerical combinations (such as OTP codes or phone numbers between 5 and 20 digits) from the remaining static. Anything failing high-entropy structural checks is discarded to guarantee only clean, relevant intelligence is passed to the user.

## Running the Application

To start the Python backend server, execute the following command from the project root:

```bash
python -m uvicorn api:app --reload
```

The FastAPI backend will bind to `localhost` on port `8000`. Once running, verify it is operational via the base health-check route at `GET /`.

To interact with the decoder, simply double-click the `WebStego/index.html` file to launch the frontend in any web browser. An online status indicator will automatically confirm communication with the backend.

If you prefer to encode or decode via command line rather than the web UI, you can use the original python module directly:

```bash
# Hide a message
python Steganography/main.py --method encode --image photo.png --msg "Secret123" --output_file hidden.png

# Reveal a message
python Steganography/main.py --method decode --image hidden.png
```
